Modular and Responsive CSS Toolkit

Getting started with ApplePie is easy. Whether you are building a simple site with a 'default' UI, or you are a master of SASS, building a new app - this toolkit will help you get up and running!

Want to get familiar?

ApplePie is open source and may be used as a starter toolkit for bootstraping and prototiping your application. You can include whole toolkit to your project or use individual ApplePie modules that may be replaced or restyled in future.

Check out docs for more info

Semantic

ApplePie encourages you to create clean & simple and completely semantic markup. Do not make up new strange class names - use language you are already familiar with and name objects as you call them in real life!

CSS Preprocessor

Want more control over appearance? Check out how to get started with Sass which will let you customize UI module by module or replace certain modules with your own styles without breaking anything globally.

Responsive friednly

ApplePie is build in a way to help create responsive interfaces and it doesn't contain responsive anti-patterns like fixed width (fixed grids) or px font-sizes.
